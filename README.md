# PersonalPortfolioSite

Simple static personal portfolio built with only HTML and CSS. Open `index.html` in a browser or use VS Code Live Server to preview.

Files:
- `index.html` — main site (hero, projects, prompts, contact)
- `resume.html` — ATS-friendly resume/CV
- `styles.css` — stylesheet
- `images/photo.svg` — placeholder profile image

Updates in this version:
- Added two more project cards: Design System Kit and Accessibility Audit
- Updated color palette to a teal / purple theme (CSS variables in `styles.css`)
- Improved `resume.html` print styles and added instructions to save as PDF

To preview locally (in VS Code):
1. Open the folder in VS Code.
2. Install the "Live Server" extension (optional).
3. Right-click `index.html` and select "Open with Live Server" or open directly in the browser.

To export the resume to PDF:
1. Open `resume.html` in your browser.
2. Press `Ctrl+P` (or File → Print).
3. Choose "Save as PDF" as the destination and save.
